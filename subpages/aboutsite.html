🎯 設計コンセプト

このサイトは、単なる情報発信の場ではなく、触って楽しい見て面白い、玉手箱のような構成としております。

JavaScriptの複雑性を排除し、純粋なCSS変数の力で、動的で美しいUXを実現。27色の「世界観」を切り替えることで、同じHTMLが全く異なる表情を持つシステムを構築しました。

🛠️ 技術スタック

フロントエンド

  • HTML5 - セマンティックで意図的な構造設計
  • CSS3 + CSS Variables - 27色のテーマシステムを完全制御
  • JavaScript - ミニマルで効率的なロジックのみ

アーキテクチャ

  • ファイル分割（タスク分担） - ページ別・機能別CSS + 共通スタイル
  • CSS動的ロード - JavaScriptで現在のURLを検出し、必要なCSSのみ読み込み
  • localStorage連携 - テーマ選択をブラウザに記憶（ローカルストレージへの一時記憶）

🎨 27色のテーマシステム

このサイトの大きな特徴は、多種類の異なる「世界観」を瞬時に切り替えられることです。

各テーマは、以下の4つのカラー変数（CSS）で構成されています：

  • --main-color - 主要なアクセントカラー
  • --balance-color - 補助的なカラー
  • --variation-color - テキスト
  • --accent-color - 背景色

これらの変数がCSS内で参照されるため、新しいテーマを追加するだけで、すべてのページが自動的に新しい色で表現されます。\n

✨ アニメーション哲学

単なる「見た目の面白さ」ではなく、UXの直感性を高めるためのアニメーションを採用。

  • 粒子メニュー - スムーズな開閉アニメーションと、階層的なナビゲーション。自動演算による均等な要素配置。片手ユーザーにも利用しやすい仕様。
  • ページ遷移 - 微妙なトランジションで、視認性を損なわない
  • ホバーエフェクト - インタラクティブな要素に対する即座の反応
  • テーマ切り替え - 世界観の変化を視覚的に体験

🚀 パフォーマンス最適化

「Asobi Lab.」 は、「退屈なWEBを面白く」することが目標です。しかし、面白さの代わりに体験が損なわれては本末転倒。

以下の工夫で、高速でスムーズな体験を実現：

  • 外部ライブラリの最小化（Pifyのみ）
  • 画像とビデオの遅延ロード
  • CSS Variables（CSS変数）の効率的な活用
  • Javascriptを用いたタスクマネージャーの導入
